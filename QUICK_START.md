## Setup in 3 Steps (Once Only)

### 1. Get Cursor IDE (Free)
- Download: [cursor.com](https://cursor.com)
- Install → Sign in with email
- You get **2000 FREE AI requests/month** forever!

### 2. Connect to GitHub
```bash
# Create GitHub account at github.com
# Create repo: the-integration-studio
# Then in Terminal:
git remote add origin https://github.com/YOUR_USERNAME/the-integration-studio.git
git push -u origin main
```

### 3. Connect GitHub to Vercel
- Go to [vercel.com](https://vercel.com)
- Import your GitHub repo
- Click Deploy - Done!

## Make Your First Edit (30 seconds)

1. **Open** Cursor
2. **Open folder**: The_integration_studio
3. **Press** `Cmd + L` (opens AI chat)
4. **Type**:
   ```
   "Change the hero heading in index-de.html to 'Transformation Studio'"
   ```
5. **Click** "Accept" → **Save** (`Cmd + S`)
6. **Deploy**:
   ```bash
   git add .
   git commit -m "My first update"
   git push
   ```

**Check website in 30 seconds - your change is LIVE! 🎉**
