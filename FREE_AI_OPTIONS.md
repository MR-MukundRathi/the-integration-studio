## 🎓 Detailed Workflow: ChatGPT + VS Code

Since ChatGPT is very popular, here's a detailed guide:

### **Setup (Once):**

1. **Install VS Code:**
   - Go to [code.visualstudio.com](https://code.visualstudio.com/)
   - Download → Install
   - Open your website folder

2. **Sign up for ChatGPT:**
   - Go to [chat.openai.com](https://chat.openai.com/)
   - Sign up with email (FREE)

### **Making Changes:**

1. **Open file** in VS Code (e.g., `index-de.html`)

2. **Copy content** (`Cmd + A` → `Cmd + C`)

3. **Go to ChatGPT** and start a new chat:
   ```
   I'm maintaining a bilingual website. Here's my German homepage file.
   I'll ask you to make changes to it.

   [Paste your file content here]
   ```

4. **Ask for specific changes:**
   ```
   Change the hero heading (currently "The Integration Studio")
   to "Leadership & Transformation Studio"
   ```

5. **ChatGPT responds** with the updated code

6. **Copy the relevant section** and paste back into VS Code

7. **Save** (`Cmd + S`)

8. **Deploy** (see QUICK_START.md)

### **Pro Tips:**

- **Create a "master prompt"** and reuse it:
  ```
  You are helping me maintain my website.
  When I ask for changes, provide ONLY the updated HTML section,
  not the entire file. Use the same style and formatting as the original.
  ```

- **Use ChatGPT Projects** (free feature):
  - Create a project called "My Website"
  - Upload key files as reference
  - ChatGPT remembers context across chats!

## 🌐 Online Code Editors (No Installation)
